## Diagrams

MPE supports rendering diagrams:
- flow charts
- sequence diagrams
- mermaid
- PlantUML
- WaveDrom
- GraphViz
- Vega
- Vega-lite
- Ditaa

Using [Code Chunk](https://shd101wyy.github.io/markdown-preview-enhanced/#/code-chunk) MPE feature, you can also render:
- TikZ
- Python Matplotlib
- Plotly
- other graphs and diagrams
