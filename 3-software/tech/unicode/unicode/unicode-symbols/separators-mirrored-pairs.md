Unicode defined these pairs that are considered mirrored glyphs of each other used in quoting.

- ⸜the⸜ ⸜omission⸝ and ⸌brackets⸍ ⸍to⸍ ⸌the⸌ ⸝effect⸝

- the ᚛ogham᚜ ᚜reversed᚛ feather mark
