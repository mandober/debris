# JS :: Abbreviations

## ECMA-262
The official ID for the ECMAScript language specification.

## HOF
Higer-order functions

## MERN
MERN Stack: MongoDB, Express, React, Node.js

## POJO
Plain-old JavaScript object (non-specialized JS object)

## TC39
Technical committee 39 (communists that govern EcmaScript standard).

## TCO
1. Tail-call optimization
2. Tail-call optimized

## PTC
Proper tail-call

## STC
Synthetic tail-call (proposal to include a keyword for signaling PTCs)
