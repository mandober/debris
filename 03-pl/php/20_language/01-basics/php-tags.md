# PHP tags

http://php.net/manual/en/language.basic-syntax.phptags.php

- long tags
  - start: `<?php`
  - end: `?>` Leave out for pure php files
- short tags *deprecated*
  - start: `<?`
  - end: `?>`
- shorthand print tags
  - `<?= expr ?>`
- ASP tags *deprecated*
  - start: `<%`
  - end: `%>`


In popular templating engines
* blade
  - `{{ expr }}`
